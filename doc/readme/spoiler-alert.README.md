# Spoiler-Alert for 朔

朔の拡張機能として働くSpoiler-Alertです。
IMG要素にシェーディングフィルタを掛け、マウスオーバー/マウスクリックに対応してフィルタを解除します。


## 使い方

* 朔ver.3.12.0以降の場合
    spoiler.min.jsと20imgspoiler.jsをwwwフォルダに入れて下さい。
* 朔ver.3.11.1以前の場合
    spoiler.min.jsと最新版朔に同梱のjquery.min.jsと20imgspoiler.legacy.jsをwwwフォルダに入れて下さい。
